### Usage:

Upload all files to your shared hosting server. Access `index.php` for the homepage and all tools via their respective PHP pages.

To build a minified version for deployment:

1. Install Node.js and npm on your system.
2. Run `npm install` in the `seo-tools` directory to install Grunt and dependencies.
3. Run `npm run build` or `grunt` to generate minified assets in the `dist/` folder.
